# 属性

# 什么是Java属性

在Java中，属性（Property）是指类中声明的变量。它们用于存储对象的状态和数据。属性可以是任何数据类型，例如整数、浮点数、字符、布尔值等等。

# 属性的分类

属性可以分为四种类型：

  1. 实例变量（Instance Variable）：实例变量是属于对象的，每个对象都有自己的一份实例变量。实例变量可以在类中的任何方法中访问，也可以通过类的对象进行访问和修改。通常在类的构造函数中初始化实例变量。

  1. 类变量（Class Variable）：类变量是属于类的，它们在所有对象之间共享。类变量可以通过类名直接访问，也可以通过对象访问。通常使用static关键字声明类变量。

  1. 常量(constant Variable): 是指一旦赋值就不能再修改的值。常量通常在程序中多次使用，因此使用常量可以提高代码的可维护性和可读性。

  1. 局部变量(Local variables): 是指在方法、构造函数或代码块中声明的变量。局部变量只在声明它的代码块内部可见，超出这个范围，它就会被销毁。局部变量在声明时必须指定类型和名称，并且可以在声明时或稍后进行初始化。

例如，下面是一个包含实例变量和类变量的示例：

public class Person {
    
    // 实例变量
    private String name;
    
    private int age;
    
    // 类变量
    private static int count;
    
    // 常量
    public static final int MAX_VALUE = 100;
    
    // 构造函数
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
        count++;
    }
    
    // getter和setter方法
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    public int getAge() {
        return age;
    }
    
    public void setAge(int age) {
        this.age = age;
    }
    
    // 类方法
    public static int getCount() {
        return count;
    }
    
    public void test() {
        int x = 1; // 声明并初始化局部变量x
        if (x > 0) {
            int y = 2; // 声明并初始化局部变量y
            System.out.println("x = " + x + ", y = " + y);
        }
        // x可以在此处访问，但是y不可以
        System.out.println("x = " + x);
    }
}

在这个例子中，Person类包含了两个实例变量name和age，以及一个类变量count。在类的构造函数中，每次创建新的Person对象时，count变量会自增1。同时，Person类还包含了getter和setter方法，用于访问和修改实例变量，以及一个类方法getCount() ，用于访问类变量count。其中MAX_VALUE是常量,不可变,test方法中声明的int x = 1 是局部变量,外部无法方法只在方法内有效,超出这个范围，它就会被销毁。 需要注意的是，局部变量在声明时必须进行初始化，否则编译器会报错。
