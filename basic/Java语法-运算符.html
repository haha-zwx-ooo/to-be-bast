# 运算符

在Java中，运算符是用来执行各种运算操作的符号或关键字。Java中的运算符分为以下几类：

  1. 算术运算符：用于执行基本的数学运算，包括加、减、乘、除、取模等，常用的算术运算符有+、-、*、/和%。

  2. 赋值运算符：用于将值赋给变量，常用的赋值运算符有=、+=、-=、*=、/=和%=等。

  3. 比较运算符：用于比较两个值是否相等或大小关系，常用的比较运算符有==、!=、<、>、<=和>=等。

  4. 逻辑运算符：用于执行逻辑运算，包括与、或、非等，常用的逻辑运算符有&&、||和!等。

  5. 位运算符：用于执行二进制位上的运算，包括按位与、按位或、异或等，常用的位运算符有&、|、^、<<、>>和>>>等。

  6. 条件运算符：用于根据条件执行不同的操作，常用的条件运算符是三目运算符，即?和:。

下面是一个简单的Java运算符的示例：

public class Main {
    
    public static void main(String[] args) {
        int a = 10;
        int b = 5;
        int c = 0;
        
        c = a + b; // 加法运算
        System.out.println("a + b = " + c);
        
        c = a - b; // 减法运算
        System.out.println("a - b = " + c);
        
        c = a * b; // 乘法运算
        System.out.println("a * b = " + c);
        
        c = a / b; // 除法运算
        System.out.println("a / b = " + c);
        
        c = a % b; // 取模运算
        System.out.println("a % b = " + c);
        
        a++; // 自增运算
        System.out.println("a++ = " + a);
        
        b--; // 自减运算
        System.out.println("b-- = " + b);
        
        boolean result = a > b; // 大小关系运算
        System.out.println("a > b = " + result);
        
        boolean bool1 = true;
        boolean bool2 = false;
        boolean bool3 = bool1 && bool2; // 逻辑与运算
        System.out.println("bool1 && bool2 = " + bool3);
        
        int num1 = 0b1010;
        int num2 = 0b1100;
        int num3 = num1 & num2; // 按位与运算
        System.out.println("num1 & num2 = " + num3);
        
        int num4 = a > b ? a : b; // 三目运算符
        System.out.println("a > b ? a : b = " + num4);
    }
}
