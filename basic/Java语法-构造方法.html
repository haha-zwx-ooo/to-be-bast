# 构造方法

# 什么是构造方法

在Java中，构造方法是一种特殊的方法，用于在创建对象时初始化对象的成员变量。每个类都可以拥有一个或多个构造方法，其名称必须与类名相同，且没有返回类型（包括void）。

当使用new关键字创建对象时，构造方法会被自动调用。如果类没有显式地定义构造方法，则编译器会默认生成一个无参构造方法。如果类定义了一个或多个构造方法，则必须显式地使用其中一个构造方法来创建对象。

构造方法可以包含任意数量的参数，参数的类型和顺序必须在方法定义中声明。在构造方法的方法体中，可以使用this关键字来引用当前对象，从而访问和修改对象的成员变量。

以下是一个简单的Java构造方法的示例：

public class Person {
    
    private String name;
    
    private int age;
    
    // 无参构造方法
    public Person() {
        this.name = "Unknown";
        this.age = 0;
    }
    
    // 带参构造方法
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 访问器方法
    public String getName() {
        return this.name;
    }
    
    public int getAge() {
        return this.age;
    }
}

public class Main {
    
    public static void main(String[] args) {
        Person person1 = new Person(); // 调用无参构造方法
        System.out.println(person1.getName() + " is " + person1.getAge() + " years old.");
        
        Person person2 = new Person("Tom", 25); // 调用带参构造方法
        System.out.println(person2.getName() + " is " + person2.getAge() + " years old.");
    }
}

在这个例子中，定义了一个名为Person的类，其中包括一个无参构造方法和一个带参构造方法。在Main类中，分别使用无参构造方法和带参构造方法创建两个Person对象，并调用访问器方法获取对象的属性值。
