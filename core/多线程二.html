# 多线程使用

# 创建线程

Java 中创建线程的方式有两种：

  1. 实现 Runnable 接口，重写 run() 方法，并创建 Thread 对象来启动线程。
  2. 继承 Thread 类，并重写 run() 方法来启动线程。

需要注意的是，使用继承 Thread 类的方式可能会受到单继承的限制，而使用实现 Runnable 接口的方式则可以避免这个问题。

public class MyRunnable implements Runnable {
    
    public void run() {
        // 线程执行的代码
    }
}

public class Main {
    
    public static void main(String[] args) {
        MyRunnable myRunnable = new MyRunnable();
        Thread thread = new Thread(myRunnable);
        thread.start();
    }
}


public class MyThread extends Thread {
    
    public void run() {
        // 线程执行的代码
    }
}

public class Main {
    
    public static void main(String[] args) {
        MyThread myThread = new MyThread();
        myThread.start();
    }
}

# 线程状态

Java线程状态包括以下几种：

  1. NEW：线程刚被创建，但还没有开始执行。
  2. RUNNABLE：线程正在Java虚拟机中执行，但它可能正在等待来自操作系统的其他资源，如处理器。
  3. BLOCKED：线程被阻止等待获取一个监视器锁，以便进入一个同步块/方法。
  4. WAITING：线程在等待某个条件（如调用了wait()方法）。
  5. TIMED_WAITING：线程在等待某个条件（如调用了Thread.sleep()方法或带有超时参数的wait()方法）。
  6. TERMINATED：线程已经执行完毕并退出。

可以使用Thread类的getState()方法来获取线程的当前状态。

# 如何启动和停止线程

启动线程通常使用Thread类的start()方法

停止线程有多种方法，以下是一些常用的方法：

  1. 使用标志位控制线程停止：在线程中定义一个boolean类型的标志位，用于控制线程的运行状态， 当标志位被设置为false时，线程执行完当前任务后退出循环，线程自然结束。
public class MyThread extends Thread {
    
    private volatile boolean flag = true; //定义一个标志位
    
    @Override
    public void run() {
        while (flag) {
            //执行任务
        }
    }
    
    public void stopThread() { //停止线程的方法
        this.flag = false;
    }
}
  1. 使用interrupt()方法中断线程：调用线程的interrupt()方法，会将线程的中断状态设置为true， 线程可以通过isInterrupted()方法检测自己是否被中断，并在适当的时候退出。
public class MyThread extends Thread {
    
    @Override
    public void run() {
        while (!isInterrupted()) {
            //执行任务
        }
    }
}
